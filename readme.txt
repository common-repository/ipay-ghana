iPay Ghana WordPress is a WordPress plugin for your [WordPress](https://wordpress.org/) websit.. It enables you receive payments online in Ghana, from your customers.

== Description ==

iPay Ghana WordPress is a WordPress plugin that enables you to receive payments online in Ghana, from your customers. It is easy to configure and use.

Visit our [get started](https://manage.ipaygh.com/xmanage/get-started) page and sign up with the easy steps provided.

== Installation ==

= Minimum Requirements =
* iPay Ghana Merchant account (login or signup [here](https://manage.ipaygh.com/xmanage/get-started) to get started)
* PHP version 5.2.4 or greater (PHP 7.2 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* You should also check [WordPress requirements](http://wordpress.org/about/requirements/) and [WooCommerce requirements](https://docs.woocommerce.com/document/server-requirements/)

= Automatic installation =

To install the iPay Ghana WordPress plugin with much ease and directly from your WordPress dashboard,
1. Go to Plugins > Add New.
2. Type "iPay Ghana WordPress" in the search box provided; from the search result, click on the "Install Now" button on the iPay Ghana WordPress plugin card to install it.
3. After installation you can configure and enable the iPay Ghana WordPress plugin under your settings dashboard.

= Manual installation =

To [manually install](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation) the iPay Ghana WordPress plugin, you must first download it, then, unzip and upload it to the `/wp-content/plugins/` directory of your webserver using your favourite FTP application.

After this stage, proceed to the Plugins page of your WordPress dashboard so as to locate and Activate the iPay Ghana WordPress plugin.

You may now visit your Settings page and configure the iPay Ghana WordPress plugin based on your preferences.
